# 그림판 프로젝트 - paintjs

# Introduction
+ <b>Canvas API</b>
  - 색 선택 하여 선(LINE) 또는 배경 채우기(FILL)
  - 슬라이드 바를 조정하여 브러쉬의 두께 조절
+ <b>downloads API</b>
  - SAVE 버튼 클릭 시 이미지 저장
 
# Stack
+ <b>HTML</b>
+ <b>CSS</b>
+ <b>JavaScript</b>
